# python redis
1. install redis client for python
    a. git clone git://github.com/andymccurdy/redis-py.git
    b. cd redis-py
    c. sudo python setup.py install 
or
    sudo pip install redis

2. install hiredis for performence. (optional but recomanded)
    pip install hiredis
